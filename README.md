# UPSC CSE 2027 — AI Study System

## Architecture
- 80,808 chunks across 30+ textbooks + PYQs + Telugu Optional
- FAISS semantic search (IndexFlatIP, 1024-dim Qwen3)
- GraphRAG: kg_entities + kg_relationships (Delta tables)
- Agent endpoint: Llama 3.3 70B with FAISS + graph_traverse tools
- Daily CA pipeline: Perplexity → Delta → FAISS → Obsidian
- Obsidian vault: spaced repetition, Dataview, Kanban

## Vault Structure
01_Current_Affairs/ — daily CA notes (auto-generated by NB6)
02_Subjects/        — subject folders (Level 0 sessions)
03_PYQs/            — PYQ bank by subject and year
04_Traps/           — trap database
05_Revision/        — spaced repetition due items
06_Answer_Practice/ — KARL evaluated answers
07_Sync/            — sync scripts

## Daily Pipeline
7:00 AM IST — NB6: Perplexity → stories → FAISS rebuild
8:00 AM IST — NB7: 6 practice modes via agent endpoint
8:15 AM IST — Vault syncs to Mac + GitHub

## Commands (Claude Code CLI)
Level 0: [topic] — start from scratch
/ask [question]  — query 80K chunk knowledge base
/compare [topic] — RAG vs GraphRAG answer side by side
/chain [entity]  — traverse knowledge graph
/today           — today's CA summary + traps
/telugu          — Telugu Optional PYQ practice
