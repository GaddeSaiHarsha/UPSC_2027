## Troubleshooting Guide

### NB6 didn't run (no stories for today)

**Symptoms:** NB7 modes return NULL, NB8 shows 0 stories
**Fix:**
1. Check NB6 job status in Databricks Jobs
2. Run NB6 manually if needed
3. NB8 will show a warning but won't fail — re-run after NB6 completes

### NB7 mode exports missing

**Symptoms:** NB8 Cell 2 shows "⬜" for mode files
**Fix:**
1. Verify NB7 ran completely (check cell 11 — export cell)
2. NB8 has fallback: it reads from `daily_practice_queue` table
3. NB8 also reads Delta tables directly for story context
4. Re-run NB7 export cell if needed

### ai_query() timeout or error

**Symptoms:** "TEMPORARILY_UNAVAILABLE" or timeout in podcast/insights generation
**Fix:**
1. Wait 2–3 minutes and re-run the failing cell
2. If persistent, check Model Serving endpoint status
3. Reduce `max_tokens` in the `modelParameters` struct
4. Try reducing input size (lower truncation limits in Cell 3)

### Obsidian files not syncing

**Symptoms:** Files in Volume but not in Obsidian vault
**Fix:**
1. Check Volume mount: `ls /Volumes/upsc_catalog/rag/obsidian_ca/Daily_Practice/`
2. Verify your sync mechanism (Volume mount, rsync, or cloud sync)
3. Check file permissions on the Volume

### daily_practice_queue table doesn't exist

**Symptoms:** Error in NB7 cell 10 about table not found
**Fix:**
```sql
CREATE TABLE IF NOT EXISTS upsc_catalog.rag.daily_practice_queue (
  queue_id STRING,

  mode6_telugu_answer STRING,
  mode7_tutor_brief STRING,
  mode8_phone_summary STRING,
  audio_script STRING,
  memory_hook STRING,
  telegram_msg_id STRING,
  obsidian_path STRING
);
```

## Customization Options

### Change podcast style/length

In NB8 Cell 4, modify the `podcast_prompt` variable:

- **Duration:** Change "Target duration: 15–20 minutes" to your preference
- **Hosts:** Rename Arjun/Priya or change their expertise areas
- **Style:** Add/remove conversation style directives
- **Sections:** Modify the `[TIMESTAMP]` structure to add/remove segments
- **Temperature:** Increase from 0.4 for more creative dialogue, decrease for factual

### Change key insights format

In NB8 Cell 5, modify the `insights_prompt` sections:

- Add new sections (e.g., "Essay Topics", "Interview Questions")
- Change flashcard count (currently 5 Q&A pairs)
- Add Telugu-specific insights section
- Modify the revision book references

### Add new practice modes

1. Add new SQL/Python cell in NB7 (after existing modes)
2. Add the mode output to `queue_data` dict in NB7 cell 10
3. Add export logic in NB7 cell 11 (export cell)
4. Add filename to `MODE_FILES` dict in NB8 Cell 1
5. NB8 will automatically pick up the new mode file

### Customize Obsidian integration

- **Tags:** Modify the YAML frontmatter in NB8 Cell 6
- **Wikilinks:** Adjust the `[[filename|display]]` format in Daily Note
- **Templates:** Add Obsidian template markers for Templater plugin
- **Dataview:** Add dataview-compatible metadata for query aggregation
