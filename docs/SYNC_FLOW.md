# Obsidian Vault Sync — End-to-End Flow

## Overview

Content generated by the Databricks morning pipeline is automatically pushed to
the local Obsidian vault on your Mac every day at **8:15 AM IST** via `launchd`.
GitHub (`NB9`) serves as a daily backup/version-history layer — **not** the
primary sync path.

## Databricks Volume Paths

| Content | Volume Path |
|---|---|
| Main vault (UPSC_2026 content) | `/Volumes/upsc_catalog/rag/obsidian_ca/UPSC_2026/` |
| Daily practice files | `/Volumes/upsc_catalog/rag/obsidian_ca/Daily_Practice/` |

Both are synced into `~/Desktop/UPSC_2027/` on your Mac.

## Sync Script Modes

```bash
# Full sync: UPSC_2026 vault + Daily_Practice (default, runs at 8:15 AM)
python3 ~/Desktop/UPSC_2027/07_Sync/sync_from_databricks.py

# Only today's CA notes (fast, ~5 seconds)
python3 ~/Desktop/UPSC_2027/07_Sync/sync_from_databricks.py --ca-only

# Only Daily_Practice folder
python3 ~/Desktop/UPSC_2027/07_Sync/sync_from_databricks.py --daily-only

# Preview without writing files
python3 ~/Desktop/UPSC_2027/07_Sync/sync_from_databricks.py --dry-run
```

## launchd Auto-Sync

The plist at `07_Sync/com.upsc.obsidian-sync.plist` triggers the full sync at
**8:15 AM IST** (= 2:45 AM UTC) daily, after NB7 (8 AM IST) has finished writing
the practice files.

Install once with:
```bash
bash ~/Desktop/UPSC_2027/07_Sync/setup_mac.sh
```

Check status:
```bash
# Verify the job is loaded
launchctl list | grep upsc

# Tail sync log
tail -f ~/Desktop/UPSC_2027/07_Sync/sync.log
```

Force a manual sync at any time:
```bash
launchctl kickstart -k gui/$(id -u)/com.upsc.obsidian-sync
```

## GitHub Backup Role

NB9 (runs at 9 AM IST) pushes a daily snapshot from Databricks to GitHub:
- `data_snapshots/{date}/` — JSON exports (mastery tracker, CA runs, etc.)
- Uses MD5 diff-skip so unchanged files are not re-committed

GitHub is **not** the primary sync path for Obsidian — it is a versioned backup.
If your Mac vault is lost, you can restore from GitHub and re-run the launchd sync.

## Troubleshooting

| Symptom | Fix |
|---|---|
| `CLI auth failed` | Run `databricks configure --profile upsc` |
| `Daily_Practice not found` | NB7 hasn't run yet today; retry after 8 AM IST |
| Stale vault after Mac sleep | Run manual sync or `launchctl kickstart` |
| Vault missing in Obsidian | Open Folder as Vault → `~/Desktop/UPSC_2027` |
